About this App The application will ask the user multiple, short questions and will provide the user three random activities that match the user's interests and desire. If the user likes the activities then the user will choose one and it will be saved in the user's history. If the user doesn't like it then they will be asked to re-random and get three more choices or restart the whole process and answer the questions one more time.
